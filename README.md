# Welcome to the FFXII Modding Guide

Welcome! This is a comprehensive guide to modding Final Fantasy XII: The Zodiac Age.

In this guide, you will find everything you need to know about the modern FFXII modding ecosystem. We will cover:

- **Modifying Game Files:** How to extract, edit, and safely apply changes to the original game files.
- **Creating Lua Mods:** How to write and use Lua scripts to change the game in real-time.
- **Modern Tools:** How to navigate and use the most up-to-date tools available in the community.
- **Creating and Distributing:** Best practices for packaging your creations and distributing them to other players.

Whether you are a beginner looking to understand the basics or an experienced user looking to adopt the newest modding standards, this guide is your starting point.

# The Insurgent's Workshop

**The Insurgent's Workshop** is a command-line tool designed for modifying Final Fantasy XII: The Zodiac Age.

**Credits:** Xeavin

### What does it do?

By default, the game's binary files are unreadable by humans. The Insurgent's Workshop solves this problem by allowing you to decompile (unpack) these binary files into readable formats like JSON or TXT.

Once the data is in a readable format, you can open it in any text editor, make your desired changes (like editing stats, names, or values), and then use The Insurgent's Workshop to compile (pack) it back into the original binary format that the game expects.

# The Insurgent's Learnable Foecraft

**The Insurgent's Learnable Foecraft** adds a configurable system for learning actions from foes while fighting them.

**Credits:** Xeavin

### What does it do?

In the vanilla game, there is no mechanic for learning enemy actions mid-battle. This mod introduces that system with full Lua configuration:

- A party member learns an action once a foe has fully cast it within their visible range.
- Every foe can have infinite learnable actions, each with its own individual chance.
- Configurable options include: which augment is required, whether status effects like Blind, Disable, or Stop prevent learning, and much more.

All parameters are loaded from `{game}/x64/scripts/config/TheInsurgentsLearnableFoecraftConfig.lua`.

# Stat Lores

**The Insurgent's Stat Lores** allows modifying existing stat lores and adding entirely new ones to the game.

### What does it do?

In the vanilla game, stat lores are augments acquired from license nodes that increase a character's stats (HP Lore, Battle Lore, Magick Lore). This mod expands the system to support all stats and multiple acquisition methods:

Supported stats:
- HP, MP, Strength, Magick Power, Vitality, Speed
- Attack Power, Defense, Magick Resist
- Evade (Parry), Evade (Weapon), Evade (Shield), Magick Evade (Shield)

Each stat lore is tied to an augment and can be acquired via:
- Unlocking a license node
- Equipping a specific piece of equipment
- Being active by default for a party member

All parameters are loaded from `{game}/x64/scripts/config/TheInsurgentsStatLoresConfig.lua`.

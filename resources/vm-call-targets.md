# VM Call Targets

**VM Call Targets** is a low-level reference for every VM call the game's script VM can invoke, including argument types and in-memory function addresses.

**Credits:** Xeavin

### What can you do with this?

EBP scripts (the map/event scripting system) work by calling into a fixed set of VM functions, movement, dialogue, camera control, item checks, and more. This spreadsheet documents every one of those calls in technical detail: argument count and type, return type, and whether it's actually used by any shipped script. It goes further than the public-facing VM Call Targets page in The Insurgent's Handbook, which lists just the name and argument count, use the Handbook page for a quick lookup and this spreadsheet when you need the full technical picture.

# Action Animations Reference

**Action Animations Reference** is a spreadsheet cataloguing the visual animation played by every spell, weapon hit, and mist action in the game.

| Credits |
| --- |
| Eochaid |

### What can you do with this?

If you plan to reassign an action to a different animation, this sheet tells you what you are actually getting. Each entry has a plain-language description of what it looks like and, more importantly, whether it is "AoE friendly," meaning the animation actually connects visually to every target it hits, not just the first one. Several familiar spells (Cure included) only animate on their first target and heal the rest silently a moment later, so this sheet catches surprises before you ship a mod. It also covers weapon hit-finish effects (the flash on a successful attack) and the animations used for Mist Actions.

# License Chip Icon Reference

**License Chip Icon Reference** is a spreadsheet that maps out `license_chip.bin`, the game file controlling which icon shows up on each node of the License Board.

| Credits |
| --- |
| Eochaid |

### What can you do with this?

If you are using The Insurgent's Toolkit to give a custom license node its own icon, this spreadsheet is your lookup table. It lists every License Board entry (weapons, armor, magick categories, augments, gambits, technicks, and more) alongside the exact values `license_chip.bin` stores for it: texture links, animation, color palette (Clut), and X/Y position. Instead of guessing which values control which icon, you find the entry you want to change by name and copy its row's values straight into the Toolkit.

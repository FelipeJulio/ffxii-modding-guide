# UI Settings Documentation

**UI Settings Documentation** is a spreadsheet documenting `uisize.json`, the game file controlling UI element sizing and positioning. Researched using The Insurgent's Toolkit.

**Credits:** Kule, Neonsquare

### What can you do with this?

`uisize.json` holds around 230 settings for cursor positions, message box padding, battle menu spacing, and more, all as raw pixel/offset values with cryptic internal names like `mCursorRightX`. This sheet explains what each setting actually visually controls, ordered to match The Insurgent's Toolkit's UI Settings editor so you can follow along top to bottom. Without it, you'd be changing values and guessing what moved.

# Gear Acquisition

**Gear Acquisition** is a spreadsheet listing every weapon, shield, helm, armor, and accessory in the game with every way a player can obtain it.

**Credits:** Eochaid

### What can you do with this?

This sheet answers one question for every piece of gear: where do you get it? It lists all 420 weapons, shields, helms, armor, and accessories, one per row, with the item's in-game rarity and each way a player can obtain it:

- **Bazaar:** the bazaar recipe that unlocks it, if any.
- **Store:** which shops sell it.
- **Treasure:** which zones have it as chest loot.
- **Enemy Drop:** which enemies drop it.
- **Steal/Poach:** which enemies you can steal or poach it from.
- **Other:** anything else, such as quests, trials, hunts, or NPC rewards.

An item often appears in several of these columns at once, since many can be bought, found, and stolen. Use it to check how vanilla acquisition works before your mod changes a drop table or a shop's stock. For the item's actual stats, cross-check Battlepack Section 13: Equipment Stats.

The sheet is community-editable, though the author asks that edits only fix mistakes rather than restructure it.

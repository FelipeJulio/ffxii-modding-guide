# Save Data Reference

**Save Data Reference** is a memory-offset map of the game's save data.

**Credits:** Eochaid

### What can you do with this?

This sheet decodes the "Global Info" section of a save file byte by byte: story flags, quest stage values and what each one means, NPC talk counters, and more. It's built for The Insurgent's Toolkit's Save Game Editor, which can view and modify essentially anything the game remembers about a player's progress. The Toolkit's own docs note that inventory changes specifically should go through the Inventory Editor instead, the Save Game Editor can technically touch inventory data too, but it's easy to break.

# Action Formulas Reference

**Action Formulas Reference** is a spreadsheet documenting all 108 damage and effect Formulas the game uses, and the individual Functions each formula calls internally.

**Credits:** Eochaid, Xeavin

### What can you do with this?

Every action in the game resolves through one of these Formulas, a named sequence of on-cast and on-hit Functions (like `MagickEvadeCheck` or `AddStatus`) that determine what actually happens when a spell or attack lands. This sheet explains, function by function, how a given Formula actually calculates its result, which goes well beyond just knowing an action's raw Power and Accuracy numbers. Every action is listed under the Formula it uses, so you can look up a spell by name and trace exactly how it works.

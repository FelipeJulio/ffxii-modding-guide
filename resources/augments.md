# Augments

**Augments** is an in-progress catalogue of every augment in the game.

| Credits |
| --- |
| Eochaid |

### What can you do with this?

An augment is the passive ability granted by an accessory or a License Board node, things like Adrenaline, Spellbreaker, or a flat HP bonus. This sheet lists all 129 augments with what they actually do, their internal ability ID (used elsewhere in game data to reference them), and community-proposed names for augments that were never given an official one in-game. If you are changing what an accessory grants, or want to understand a License Board effect before editing it, this is the reference.

# Augments

**Augments** is an in-progress catalogue of every augment in the game.

**Credits:** Eochaid

### What can you do with this?

An augment is the passive ability granted by an accessory or a License Board node, things like Adrenaline, Spellbreaker, or a flat HP bonus. This sheet lists all 129 augments with what they actually do, their internal ability ID (used elsewhere in game data to reference them), and community-proposed names for augments that were never given an official one in-game. Many entries repeat a generic effect like "Battle Lore" or "Magick Lore" at several power tiers (30/50/70/100), each with its own ID, since gear and licenses reference augments by number, not by name. If you are changing what an accessory grants, or want to understand a License Board effect before editing it, this is the reference.

The Insurgent's Toolkit's own Battlepack Editor page for augments is short and mostly just points here for the real detail, this spreadsheet (not the Toolkit's own UI) is the actual reference for augment IDs and effects.

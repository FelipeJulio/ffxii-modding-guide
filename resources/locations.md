# Locations

**Locations** is the master index of every location in the game and which underlying files it loads.

**Credits:** Xeavin

### What can you do with this?

Before you can export or target the right file in The Insurgent's Toolkit's ARD or EBP editors, you need to know which `.ard` and `.ebp` file your current location actually uses, since many locations share the same file. This spreadsheet is that lookup: location name, region, and the exact `.ard`/`.ebp`/map control file names behind it. It's referenced directly from multiple Toolkit editor pages as the way to figure out the correct export path.
